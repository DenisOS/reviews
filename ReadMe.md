# Reviews Node Apps

# Usage

### Starting reviews locally from /reviews

    node bin/server

*Now you can visit http://localhost:8080 to view your application*

### Deploy helloworld to nodejitsu

    jitsu deploy

*You will now be prompted for a `subdomain` to deploy your application on*
